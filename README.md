## Description

The **Sales Report for SureCart** plugin allows you to generate **CSV sales reports** directly from your WordPress dashboard.  

With this plugin, you can export sales data for:
- Individual products  
- Specific SureCart Price IDs  

Reports are pulled from the **SureCart API** and formatted into clean CSV files that you can open in Excel, Google Sheets, or any spreadsheet tool.

---

## Features

- Export CSV reports by **product** or **Price ID**  
- Lightweight and easy to use  
- Works seamlessly with the SureCart API  
- Designed for **store owners** and **accountants** who need quick access to product sales data  

---

## Installation

1. Download the plugin `.zip` file.  
2. Upload it to your WordPress site via **Plugins → Add New → Upload Plugin**.  
3. Activate the plugin through the **Plugins** menu in WordPress.  
4. Navigate to the plugin settings under **SureCart → Sales Report**.  

---

## Usage

1. Go to **Tools→ SureCart Export** in your WordPress dashboard.  
2. Choose which product to generate a report by:
   - **Product**  and if you want a specific price-id 
3. Click **Export CSV** to download your sales data.

## Frequently Asked Questions

A: No. This plugin requires SureCart to be installed and configured.

A: The plugin generates a `.csv` file that can be opened in Excel, Numbers, or Google Sheets.
